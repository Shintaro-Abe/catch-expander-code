# 要求定義 — サブエージェントのプロンプト・出力記録

> 関連: [`design.md`](./design.md) / [`tasklist.md`](./tasklist.md)

---

## 背景・目的

エージェント実行時にリサーチャー・ジェネレーター・レビュアーが使用したプロンプトと
出力結果を記録・閲覧できるようにする。

主な目的:
1. **プロンプトデバッグ**: 生成品質が期待通りでなかった実行を遡って原因調査できる
2. **プロンプト改善**: 複数実行の入出力を比較してプロンプトを最適化できる
3. **ナレッジ水平展開**: LLM エージェントのプロンプト記録パターンを他プロジェクトの実装例として活用する

---

## ユーザーストーリー

- 管理者として、特定の実行のリサーチャー・ジェネレーター・レビュアーに送ったプロンプト全文をダッシュボードで確認したい
- 管理者として、各サブエージェントの出力（レスポンス）をダッシュボードで確認したい
- 管理者として、生成結果が期待通りでなかった実行について、どのような入力でどんな出力が生成されたかを事後に調査したい

---

## 受け入れ条件

- [ ] 実行詳細画面（`/executions/:id`）にサブエージェントの入出力を表示するセクションがある
- [ ] リサーチャー・ジェネレーター・レビュアーごとにプロンプトと出力を確認できる
- [ ] リサーチャーは複数ステップを並列実行するため、ステップごとに個別表示する
- [ ] プロンプト・出力は長大になるため折りたたみ（展開ボタン）で表示する
- [ ] データが存在しない実行（記録機能追加前の実行）は「データなし」を表示する

---

## 機能要件

### 記録対象

| サブエージェント | プロンプト | 出力 |
|----------------|-----------|------|
| リサーチャー（ステップ×N） | researcher.md + 調査指示 | 調査結果 JSON |
| ジェネレーター | generator.md + リサーチ結果 + プロファイル | 生成物 JSON |
| レビュアー（ループ×M） | reviewer.md + 成果物 + ソース | レビュー結果 JSON |

### ストレージ設計

プロンプト・出力はサイズが大きいため（50〜200KB）、**S3 に保存**する。
DynamoDB には S3 キーのみを記録する。

```
S3 キー構造:
  prompts/{execution_id}/researcher_{step_id}.json
  prompts/{execution_id}/generator_0.json
  prompts/{execution_id}/reviewer_{loop}.json
```

各 JSON ファイルの構造:
```json
{
  "subagent": "generator",
  "index": 0,
  "prompt": "...(全文)...",
  "output": "...(全文)...",
  "recorded_at": "2026-05-05T10:00:00Z"
}
```

### ダッシュボード表示

実行詳細画面の既存タイムラインの下に「サブエージェント入出力」セクションを追加する。

```
┌──────────────────────────────────────────────┐
│ サブエージェント入出力                         │
├──────────────────────────────────────────────┤
│ ▼ リサーチャー                                 │
│   ステップ 1: Terraform の基本概念             │
│   [プロンプトを表示 ▶]  [出力を表示 ▶]        │
│   ステップ 2: IaC の比較                       │
│   [プロンプトを表示 ▶]  [出力を表示 ▶]        │
├──────────────────────────────────────────────┤
│ ▼ ジェネレーター                               │
│   [プロンプトを表示 ▶]  [出力を表示 ▶]        │
├──────────────────────────────────────────────┤
│ ▼ レビュアー（ループ 1/2）                     │
│   [プロンプトを表示 ▶]  [出力を表示 ▶]        │
└──────────────────────────────────────────────┘
```

---

## 非機能要件

- S3 ファイルは実行完了から **5年後に自動削除**（DynamoDB TTL と統一）
- プロンプト・出力の取得は実行詳細画面を開いたときに **オンデマンドで取得**する（一覧表示時には取得しない）
- S3 へのアクセスは ECS タスクロール・Lambda ロール経由（パブリックアクセス禁止）

---

## 制約・スコープ外

- 実行一覧画面へのプロンプト表示: スコープ外
- プロンプトの検索・全文検索: スコープ外
- 複数実行間の diff 表示: スコープ外（将来拡張）
