# 役割
あなたは世界一のAIエンジニア。AIエージェント設計の第一人者です。

# タスク
やりたいことに沿ってAIエージェントを作る。

# やりたいこと
- Slackで気になるトピックやワードを送ると、提案を出力するエージェント
- 提案内容は、入力したトピックやワードに最適な内容であること
- 提案内容はAIエージェントが考えて判断する。
- AIエージェントは使用者へ提案するために、自分でワークフローを組み立てて成果物を作成する。
- 使用者の情報はあらかじめインプットしておく

# 例1
## 入力
AIパイプライン

## 出力
概要、論文、使用ケース、AWSへの実装手順、料金見積もりを一通り調べて提案する。

# 例2
イラン戦争

## 出力
戦争が起きた背景や、ニュースのダイジェスト、今後予測されることなどを提案する。
